# 💍 Wedding Invitation Admin System

A complete admin dashboard for managing unlimited wedding invitation clients without modifying code.

## Features

✅ **Admin Dashboard** - Manage all clients from one place  
✅ **Add Clients** - Simple form to add new clients  
✅ **Generate Links** - Automatic unique invitation links  
✅ **Track Revenue** - Monitor payments and earnings  
✅ **Dynamic Invitations** - Each client gets custom invitation  
✅ **No Code Changes** - Everything through admin panel  
✅ **Unlimited Clients** - Scale to any number of clients  

## How It Works

1. **Open admin.html** - Your admin dashboard
2. **Add Client** - Fill in couple details, wedding info, photos
3. **Generate Link** - Get unique invitation URL
4. **Share with Client** - Send link to couple
5. **Client Views** - Their custom invitation loads automatically

## File Structure

- `admin.html` - Admin dashboard
- `admin-style.css` - Admin styling
- `admin-script.js` - Admin functionality
- `invitation.html` - Client invitation template
- `invitation-style.css` - Invitation styling
- `invitation-script.js` - Invitation functionality

## Usage

### Add New Client
1. Go to "Add New Client" tab
2. Fill in all details
3. Click "Create Client & Invitation"
4. Copy the generated link
5. Share with client

### View Client Details
1. Go to "All Clients" tab
2. Click "View" on any client
3. Copy their invitation link

### Track Revenue
1. Dashboard shows total clients
2. Total revenue from all clients
3. Upcoming weddings count
4. Payment status tracking

## Data Storage

All client data is stored in browser's localStorage. This means:
- ✅ Data persists across sessions
- ✅ No server needed
- ✅ Completely private
- ⚠️ Back up data regularly (use Export button)

## Backup & Export

1. Go to Settings
2. Click "Export All Data"
3. JSON file downloads
4. Keep safe backup
